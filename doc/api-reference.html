    HNE
    HNE.CLI

    following module is intended to be a command line interface that will include a help function and eventually capabilities for adding/removing emails and keywords

    HNE.HNStories

    This module pulls a list of the n top news stories from the HN firebase directory of Top News Stories, it then checks each news story title for ‘Elixir’ or ‘Erlang’

    HNE.Mailer

    Module sets up mailing config as well as function

    HNE.StoryId

    This module keeps track of all story ids that have been checked in the past in a csv in the root folder
